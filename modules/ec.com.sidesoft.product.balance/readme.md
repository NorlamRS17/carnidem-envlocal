Descripcion
El presente Modulo sirve para confirmar los pesos de la ventana de Albaran Proveedor y la ventana de Movimiento entre almacenes.
Se puede realizar toma de pesos de forma manual o con una balanza conectada.
Para poder hacer uso de la balanza se necesita parametrizar las Urls en la ventana de Servicio de Hardware Mnager.
Todo el funcionamiento inicia Cuando se pulsa en el boton de confirmaciones en las ventanas ya mencionadas
