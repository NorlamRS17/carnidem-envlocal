# LocalizacionOpenbravo, FacturaElectronicaOffline_Funcionalidad, OpenbravoEstandar18Q3.5

## Descripción

El presente modulo esta diseñado para interactuar con otras partes del sistema, para realizar transacciones financieras, devoluciones o reembolsos autorizados. Su objetivo principal es procesar o cargar datos específicos relacionados a reembolsos autorizados en base a los registros dentro de la Factura(Proveedor) dentro del ERP de Openbravo.
